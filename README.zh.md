## 技术栈

- Next.js 15
- shadcn/ui
- Tailwind CSS
- TypeScript

## 🚀 快速开始

```bash
git clone https://github.com/cefeng06/Ghibli-Shadcn-Theme.git
cd Ghibli-Shadcn-Theme
pnpm install
pnpm run dev
```
